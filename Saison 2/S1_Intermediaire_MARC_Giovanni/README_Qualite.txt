Optimisations réalisées :

1. Utilisation de variables CSS (`:root`) pour centraliser les couleurs, espacements et rayons de bordure, facilitant la maintenance, améliorant la lisibilité du code et réduisant la duplication.

2. Utilisation de CSS Grid avec `repeat(auto-fit, minmax(...))` afin de créer une grille responsive automatique, s’adaptant dynamiquement à la taille de l’écran sans nécessiter de multiples media queries.

3. Mise en place du système `localStorage` pour sauvegarder les préférences utilisateur (thème et contenu), permettant de restaurer automatiquement l’état de l’application après rechargement de la page.

4. Amélioration de l’accessibilité et de l’expérience utilisateur grâce à l’ajout de transitions CSS fluides et à la gestion des interactions clavier (`focus-visible`) pour une navigation plus intuitive.
